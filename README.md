# 🐃 Blog de Bubalinocultura no Paraná

Este é um projeto de site/blog desenvolvido em HTML e CSS com o objetivo de apresentar informações sobre a criação de búfalos (bubalinocultura) no estado do Paraná.

O site aborda temas como manejo, alimentação, produção de leite e carne, além de boas práticas no campo.

---

## 🌐 Demonstração

Abra o arquivo `index.html` em seu navegador para visualizar o site.
